# Tower Game Project

## Overview
Welcome to the Tower Game Project! This game is designed using Kotlin and it brings a unique twist to the classic tower defense genre. Players must strategically build towers to fend off waves of enemies, utilizing various strategies to achieve victory.

## Features
- **Multiple Tower Types**: Choose from a variety of towers, each with unique abilities.
- **Challenging Levels**: Progress through increasingly difficult stages.
- **Enemy Variety**: Different enemy types to test your strategies.
- **Kotlin-based**: Built with best practices of Kotlin for better performance.

## Installation
To get started with the Tower Game, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/androminor/BastionArmor.git
   cd BastionArmor
   ```
2. Ensure you have Kotlin installed on your system. Refer to the [Kotlin installation guide](https://kotlinlang.org/docs/installation.html) for assistance.
3. Run the project:
   ```bash
   ./gradlew run
   ```

## Gameplay
### Objective
Defend your base from enemy waves by building and upgrading towers.

### Controls
- **Mouse Click**: Select and place towers on the game map.
- **Keyboard Shortcuts**: Use shortcuts to manage in-game actions quickly.

### Scoring
Earn points by defeating enemies, which can be used to buy upgrades.

## Technologies Used
- **Kotlin**: Programming language for game development.
- **Gradle**: Build automation tool for managing dependencies.
- **LibGDX**: Framework used for game development.
